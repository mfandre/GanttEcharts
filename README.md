# GanttEcharts
Gantt Chart using echarts

# Folders
- gantt-angular-npm (angular app example)
- gantt-angular-sandbox (angular app to sandbox)
- gantt-echarts-lib (angular lib to create npm package)
- gantt-html-js (html+js version)
